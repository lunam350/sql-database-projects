# SQL Database Projects

This repository contains SQL projects developed during my Software Development studies using MySQL.

## Project: Electronic Technologies Database

This project consists of the design and implementation of a relational database for a fictional company called **Electronic Technologies**.

### Technologies used
- SQL
- MySQL
- MySQL Workbench

### Concepts applied
- Relational database design
- Primary keys and foreign keys
- Table relationships
- Data structure for orders, customers and products

### Files included
- `reto2_electronic_technologies.sql` → SQL script to create the database structure
- `Mi primer base de datos.mwb` → MySQL Workbench database model

### Database entities
The database includes tables such as:

- store
- product
- client
- payment_method
- purchase_order
- order_detail

This project demonstrates basic database design and SQL implementation.
